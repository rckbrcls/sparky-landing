# Documentation

This repository is a small static Next.js landing and legal site for the Sparky iOS app. Documentation is intentionally compact so the useful project details stay easy to find and maintain.

## Guides

- [Architecture](architecture.md) - technical structure, rendering model, component responsibilities, and current limitations.

## Primary Reference

Use the root [`README.md`](../README.md) first for setup, scripts, routes, launch notes, and project status.

## Documentation Boundaries

The project currently has no API, database, authentication flow, deployment provider configuration, or contribution workflow. Separate docs for those areas should only be added if the codebase grows to include them.
